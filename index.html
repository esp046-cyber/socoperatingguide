SOC Operator Role & Responsibilities

Primary Duties

  • 24/7 monitoring of security events and alerts
  • Initial triage and investigation of security incidents
  • Threat detection using SIEM and security tools
  • Documentation of security events and incidents
  • Escalation of critical threats to senior analysts
  • Maintaining security infrastructure uptime
Penalties: Fines up to PHP 5 million and imprisonment of 1-6 years for DPA violations. SOC operators must ensure all personal data handling is logged and protected.

PowerShell Security Scripts

# Check suspicious scheduled tasks Get-ScheduledTask | Where-Object { $_.TaskPath -notlike "\Microsoft*" } # Find recently modified executables Get-ChildItem C:\Windows\System32\*.exe | Where-Object {$_.LastWriteTime -gt (Get-Date).AddDays(-7)} # List all listening ports Get-NetTCPConnection | Where-Object { $_.State -eq "Listen" }

SOC Playbooks

Phishing Email Response

Step 1: Verify the reported email in email gateway
Step 2: Extract indicators (sender, links, attachments)
Step 3: Check if links/attachments are malicious (VirusTotal)
Step 4: Search for other users who received same email
Step 5: If malicious, quarantine all instances
Step 6: Block sender domain and URLs in gateway
Step 7: Notify affected users, document in ticket

Malware Infection Response

CRITICAL ACTIONS:
1. Isolate infected endpoint from network immediately
2. Do NOT shut down (preserve RAM for forensics)
3. Capture memory dump if trained
4. Collect EDR telemetry and logs
5. Identify patient zero and lateral movement
6. Escalate to L2 for deep forensics
7. Reimage system after IOC extraction

Ransomware Detection

INDICATORS: - Mass file encryption (.locked, .encrypted) - Ransom notes (README.txt) - Spike in file modifications - Disabled AV/EDR agents - Shadow copy deletion IMMEDIATE ACTIONS: 1. Disconnect affected systems 2. Identify ransomware variant 3. Preserve encrypted files 4. Contact cyber insurance 5. Report to CERT-PH and NPC

Philippine Compliance Requirements

Data Privacy Act of 2012 (RA 10173)

  • Implement security measures to protect personal data
  • Notify NPC of data breaches within 72 hours
  • Notify affected individuals as soon as possible
  • Appoint Data Protection Officer (DPO)
  • Conduct Privacy Impact Assessments
  • Maintain audit logs for personal data processing
